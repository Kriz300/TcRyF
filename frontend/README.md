# APP Para visualizar y filtrar tablas

APP desarrollada con React y JavaScript.

### Ejecución y Comandos 🔧
* Ejecución local
```
npm run dev
```

* Dejar la ejecución en segundo plano
```
npm install pm2 -g

pm2 start "npm run dev" --name app_name

```

## Construido con 🛠️

* **Node.js**
* **React**
* **Axios**
* **React Bootstrap**
